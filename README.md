# Expense Tracker

A simple Java Swing application to track expenses.

## Project Structure

- `src/`: Java source files.
- `bin/`: Compiled class files.
- `build.ps1`: PowerShell script to build and run the application.

## Prerequisites

- Java Development Kit (JDK) installed and available in your PATH.

## How to Run

Open a PowerShell terminal in the project directory and run:

```powershell
.\build.ps1
```

This script will:
1. Compile the source code into the `bin/` directory.
2. Launch the `ExpenseTrackerGUI` application.
